# House_Price_Prediction

House Price Prediction Using Linear Regression
A machine learning project that predicts house prices using the Ames Housing Dataset.
The model uses features like living area, bedrooms, bathrooms, total rooms, and neighborhood (label-encoded) to estimate house sale prices.
